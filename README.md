# apt-select

>  Linux 快速换源工具

- [ ] 自动检测系统版本
- [ ] Ubuntu 换源
- [ ] CentOS 换源

## 为什么会有这个项目?

作者喜欢折腾, 经常需要重装系统, 所以日常换源.



## 使用方法

```sh
cd ./apt-select

chmod +x ./apt-select.sh

./apt-select.sh ali
```
